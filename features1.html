Fetch 3D model from a remote database

  • Android Application

  • SketchUp plugin

  • Part: Mobile Application
  • Technology: Java with the Android API

With SIG-AR, your Android device can :

  • fetch 3D models from a remote database
  • display 3D models on assigned GPS coordinates
  • edit on-site your models geographical information to handle GPS errors, or to put your creation at the best place.

    • Part: Google SketchUp Plugin
    • Technology: Ruby, HTML/JavaScript/CSS/PHP and PostgreSQL

    SIG-AR is also a complete stand-alone suite: built as a Google SketchUp plugin and an Android app, it gives you complete control of the whole workflow from the creation of your model, to the on-site visualization.
